# Agent Instructions

## Agent skills

### Issue tracker

Issues and PRDs live in GitHub Issues for `jessewalberg/skills`; `Mission Control` is the cross-repo Project view. See `docs/agents/issue-tracker.md`.

### Triage labels

Use canonical GitHub labels plus exactly one `status:*` label per active issue. See `docs/agents/triage-labels.md`.

### Domain docs

Use the repo's `CONTEXT.md` and ADRs in `docs/adr/` unless `docs/agents/domain.md` says otherwise.
